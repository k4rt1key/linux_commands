# Command : lscpu

Listing all information about CPU

```bash
lscpu
lscpu -J # Json output
```

### Output information

**Architecture**
- Architecture
- CPU op-mode(s)
- Byte Order
- Address sizes

**CPU Information**
- CPU(s)
- On-line CPU(s) list
- Thread(s) per core
- Core(s) per socket
- Socket(s)
- NUMA node(s)
- Vendor ID
- CPU family
- Model
- Model name
- Stepping
- CPU MHz
- CPU max MHz
- CPU min MHz
- BogoMIPS
- Virtualization

**Cache Information**
- L1d cache
- L1i cache
- L2 cache
- L3 cache

**NUMA Information**
- NUMA node0 CPU(s)
